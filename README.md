# Empower Mind Counselling and Therapy (EMCT) Website

## Overview
This is the official website for **Empower Mind Counselling and Therapy (EMCT)**, a non-profit organization focused on promoting mental health awareness and providing accessible counseling and therapy services, particularly for young people in Malawi and beyond. The website includes sections for booking appointments, displaying the latest news, and a brief introduction to the organization's mission.
